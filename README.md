# Data Project Catalog - Interactive Dashboard

Interactive dashboard for browsing and filtering data projects.

## Features
- Search across project names, descriptions, and owners
- Filter by Asset Type (Data Collection, Workspace)
- Filter by Status (Active, Inactive, To Be Created)
- Filter by Access Level (Owner, Writer, Reader, Discover, Admins)
- Filter by Category (AI/ML, Analytics, Infrastructure, Quality, External, Product)
- Real-time stats updates
